使用hexo在GitHub搭建个人博客

发表于 2016-02-18   |   分类于 其它   |  

hexo是一款快速、简洁、高效的博客框架，底层基于node.js，编写博客采用markdown语法。github pages免费提供静态网页托管的服务，容量为
300M，对于一个博客来说，容量已经足够了，但其中也存在一个问题，github毕竟是一个国外的网站，国内访问速度有限。
hexo在升级中很多地方发生了变动，网上教程很多都是针对老版本的，自己的安装部署的时候参照老教程踩了一些坑，因此将具体的步骤记录下来，以飨读者：
